Section 01

Overview & Getting Started

SetViz is a browser-based interactive tool for learning set theory concepts through dynamic Venn diagram visualisations. No installation, no registration, no cost — open the link and start learning immediately.

🌐
Access SetViz Open any modern browser (Chrome, Firefox, Safari, Edge) and go to https://ciksuriyati.github.io/SetViz/ — works on desktop, tablet, and smartphone.
1
Open the URL No download or login needed. The page loads instantly.
2
Choose a stage from the left panel Start with Stage 0 (Orientation) or jump to any concept you need.
3
Click an operation button The Venn diagram shades instantly to show exactly which region represents that operation.
4
Read the teaching note and symbol Each operation shows a contextual explanation and its formal mathematical notation.

Section 04

Operation Buttons

Each stage shows a row of operation buttons at the top of the main panel. Clicking a button instantly shades the corresponding region in the Venn diagram.

Choose an operation

B

Section 07

All 7 Learning Stages

SetViz organises content into seven progressive stages, from the basic idea of a set all the way to De Morgan's Laws.

Stage Title Operations Available What You Learn
0 Orientation — What a set is; the universal set U as the bounding rectangle
1 One Set (A) AA′ Single set and its complement; everything outside A
2 Two Sets Overlapping ABA′B′A∪BA∩B Union, intersection, complements with two overlapping circles
3 Mutually Exclusive ABA∪BA∩B=∅ Disjoint sets; why A∩B=∅ when circles don't touch
4 Set Differences A\BB\AAΔB Set difference and symmetric difference (crescents)
5 Subset / Superset A⊂BA=BA⊄Bdisjoint Proper subsets, equal sets, non-subsets — diagram repositions dynamically
6 Complements & De Morgan A′B′(A∪B)′(A∩B)′ De Morgan's Laws: (A∪B)′=A′∩B′ and (A∩B)′=A′∪B′, visualised
Section 08

Quick-Start Workflow

Follow this sequence for first-time use or classroom demonstration.

1
Open Stage 0 — Orientation Read the teaching note. Observe the empty bounding rectangle — this is the universal set U. No circles appear yet.
2
Move to Stage 1 — One Set (A) Click A to shade circle A gold, then click A′ to shade its complement. Compare the two. Check the Symbol Notation badge for the formal expression.
3
Stage 2 — Work through all 6 operations Click A, B, A′, B′, A∪B, A∩B one by one. After each click, look at the shaded region and read the operation description below. Focus on the difference between ∪ and ∩.
4
Stage 3 — Compare with disjoint circles Click A∩B here and compare with Stage 2. The empty-set badge reinforces why A∩B=∅ when sets don't overlap.
5
Stage 6 — Verify De Morgan's Laws Click (A∪B)′ and note which region is shaded. Then click (A∩B)′. Cross-reference with your textbook to confirm the logical equivalences visually.
🎓
For lecturers SetViz works well as a live demonstration tool during lectures. Open it on the projector, select operations as you explain each concept, and let students interact on their own devices simultaneously. No setup required.
