About SetViz

An interactive learning tool for set theory concepts through visualizations

What is SetViz?

SetViz is a static web app for learning set theory concepts through interactive visualizations. We focus on understanding basic set concepts, operations, and relationships through our 3-stage learning approach. Built to mirror the simplicity of ProbViz.

Our Mission

To make set theory accessible and understandable through visual learning. We believe that seeing mathematical concepts in action helps build deeper understanding than abstract notation alone.

Learning Approach

🎯 Stage 1: One Set

Learn what a set is and understand the basic concept of set A and its complement A′

🔄 Stage 2: Two Independent Sets

Explore relationships between two separate sets with union, intersection, and complements

🔗 Stage 3: Set Relationships

Discover different relationships: Independent, Overlapping, and Mutually Exclusive sets

How to Use SetViz

  1. Start Learning: Click "Learn Basic Concepts" to begin your journey
  2. Follow the Stages: Progress through our 3-stage learning system
  3. Interactive Exploration: Click operation buttons to see visual highlights
  4. Explore Relationships: Use the relationship toggle to see different set types
  5. Visual Understanding: Watch how areas highlight to understand each concept

What You'll Learn

  • Basic Set Concepts: Understanding what sets are and how they work
  • Set Notation: Learning symbols like A, A′, ∪, ∩, and their meanings
  • Set Operations: Union (∪), Intersection (∩), Complement (′), and Set Difference
  • Set Relationships: Independent, Overlapping, and Mutually Exclusive sets
  • Visual Intuition: Building understanding through interactive visualizations
